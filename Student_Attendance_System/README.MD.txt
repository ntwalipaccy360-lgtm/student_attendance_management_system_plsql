## 📌 Project Description
This project is a Student Attendance Management System developed using Oracle SQL and PL/SQL.  
It is designed to replace manual attendance tracking with an automated database system that improves accuracy, efficiency, and reporting.

---

## 🎯 Problem Statement
Manual attendance systems in schools are inefficient and prone to errors. This system solves this problem by storing attendance data in a relational database and using PL/SQL to automate calculations and reporting.

---

## 🗂 Database Tables
The system is built using the following tables:

- Students
- Courses
- Lecturers
- Attendance
- Course_Assignment

---

## ⚙️ PL/SQL Features Implemented

### 1. Stored Procedure
- MarkAttendance
- Used to automatically insert attendance records

### 2. Function
- GetAttendancePercentage
- Calculates attendance rate for each student

### 3. Anonymous Blocks
- Used to test procedure and function execution

### 4. Window Functions
- RANK() function
- Used to rank students based on attendance performance

## 📊 System Features
